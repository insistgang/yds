+================================================================+
|                         检测层 (detector)                       |
|                                                                 |
|   +--------------------------------------------------------+    |
|   |  YOLOv11 (PyTorch)                                     |    |
|   |  model: yolo11n.pt                                     |    |
|   |  conf >= 0.25, imgsz=640                               |    |
|   |                                                          |    |
|   |  P0 四类:                                                |    |
|   |    [0] blind_road_occupied  盲道占用                    |    |
|   |    [1] stairs                台阶                      |    |
|   |    [2] ramp                  坡道                      |    |
|   |    [3] road_obstacle         路面障碍                  |    |
|   |                                                          |    |
|   |  COCO 冷启动代理 (诚实标注):                             |    |
|   |    bicycle/bench/person -> road_obstacle               |    |
|   |    stairs/ramp/blind_road 需自定义训练                  |    |
|   |                                                          |    |
|   |  异常降级: 单帧失败返回空 FrameDetections                |    |
|   |  预留后端: pytorch | onnx | tensorrt (engine字段)      |    |
|   +--------------------------------------------------------+    |
|                           |                                     |
|                           v                                     |
|   +--------------------------------------------------------+    |
|   |  单帧推理延迟: <100ms                                    |    |
|   |  FPS 目标: >=30                                          |    |
|   |  benchmark.py 记录 pre/infer/post 耗时                    |    |
|   +--------------------------------------------------------+    |
+================================================================+
                |
                | FrameDetections
                v
+================================================================+
|                      事件层 (event_builder)                     |
|                                                                 |
|   +--------------------------------------------------------+    |

|   |  内存清理    TTL 过期自动删除 _states                   |    |
|   |                                                          |    |
|   |  优先级排序:                                             |    |
|   |    stairs(100) > road_obstacle(80) >                   |    |
|   |    blind_road(70) > ramp(60)                           |    |
|   |                                                          |    |
|   |  P1 硬隔离: traffic_red/crosswalk 直接过滤              |    |
|   +--------------------------------------------------------+    |
|                           |                                     |
|                           v                                     |
|   +--------------------------------------------------------+    |
|   |  DetectionEvent                                          |    |

|   |    - priority, bbox, distance_m, direction              |    |
|   |    - source_frame_ids                                   |    |
|   +--------------------------------------------------------+    |
+================================================================+
                |
                | DetectionEvent
                v
+================================================================+
|                      语义层 (semantics)                         |
|                                                                 |
|   +--------------------------------------------------------+    |
|   |  模板化中文提示 (<50字)                                  |    |
|   |                                                          |    |
|   |  "前方盲道被占用，请注意绕行"                            |    |
|   |  "前方有台阶，请减速"                                    |    |
|   |  "前方有坡道，位于右侧"                                  |    |
|   |  "前方有障碍，请注意避让"                                |    |
|   |                                                          |    |
|   |  方向: 前方/左前方/右前方/左侧/右侧                     |    |
|   |  距离: 前方X米 (不稳定时省略)                           |    |
|   |  明确、可执行、无不确定措辞                             |    |
|   +--------------------------------------------------------+    |
|                           |                                     |
|                           v                                     |
|   +--------------------------------------------------------+    |
|   |  ALL_TEMPLATE_TEXTS (28 条)                             |    |
|   |  启动预加载至 TTS 缓存                                  |    |
|   +--------------------------------------------------------+    |
+================================================================+
                |
                | 中文提示文本
                v
+================================================================+
|                     Pipeline 仲裁层                             |
|                                                                 |
|   +--------------------------------------------------------+    |
|   |  低信息密度原则: 一帧只播最高优先级事件                  |    |
|   |  audio_cooldown_sec: 同一标签秒级冷却                   |    |
|   |  播报失败 pop 冷却记录，下次可重试                      |    |
|   +--------------------------------------------------------+    |
+================================================================+
                |
                v
+================================================================+
|                      音频层 (audio_manager)                     |
|                                                                 |
|   +--------------------------------------------------------+    |
|   |  默认音色: presenter_female                              |    |
|   |  默认模型: speech-2.8-hd                                 |    |
|   |  缓存目录: ~/.cache/linkable_edge/tts_minimax/           |    |
|   |  缓存键: SHA256(text+voice+speed+emotion+model)         |    |
|   +--------------------------------------------------------+    |
|                           |                                     |
|                           | 启动预加载 ALL_TEMPLATE_TEXTS      |
|                           v                                     |
|   +--------------------------------------------------------+    |
|   |  全局音频锁 threading.Lock()                             |    |
|   |  防止多事件音频重叠                                     |    |
|   +--------------------------------------------------------+    |
|                           |                                     |
|                           | 降级链: 失败时绝不静默             |
|                           v                                     |
|   +--------------------------------------------------------+    |
|   |  1. MiniMax TTS API -> mp3 缓存 -> mpg123/aplay 播放   |    |
|   |  2. pyttsx3 (本地离线 TTS)                              |    |
|   |  3. print (文本兜底，保证不静默)                        |    |
|   +--------------------------------------------------------+    |
+================================================================+

|   |  输出: benchmark.json + benchmark.csv                   |    |

+================================================================================+
|                          AGENTS.md 硬约束                                       |
+================================================================================+

  [OK] 输入 fallback: CSI -> USB -> 视频 -> 图片
  [OK] 音频降级链: MiniMax -> pyttsx3 -> print
  [OK] 全局音频锁: threading.Lock()
  [OK] 启动预加载: ALL_TEMPLATE_TEXTS
  [OK] 低信息密度: 一帧只播最高优先级
  [OK] 隐私保护: 默认不存原始帧 (save_interval_sec=0.0)
  [OK] P1 硬隔离: traffic_red/crosswalk 不可触发
  [OK] 性能基准: JSON/CSV 输出
  [OK] 结构化事件: 不上传原始图像
