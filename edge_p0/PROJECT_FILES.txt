LinkAble Edge P0 项目文件说明
==============================

根目录文件
----------

.gitignore              Git 版本控制忽略规则（排除 .pyc、缓存、数据集等）
ARCHITECTURE.txt        系统架构图（ASCII 绘制，含输入层/检测层/事件层/语义层/音频层/上报层/性能基准）
PROJECT_ANALYSIS.md     项目早期分析文档（2026-04-29，部分被 PRD v2.2 替代，留档备查）
pyproject.toml          Python 包配置文件（包名 linkable-edge-p0，版本 0.1.0，Python>=3.10）
README.md               项目主文档（定位、快速开始、AGENTS.md 合规表、提交材料清单）
train_yolo.py           YOLO 训练脚本（epochs=100, imgsz=640, batch=16, AdamW 优化器）


核心目录
--------

src/linkable_edge/          核心源码（17 个 Python 文件）
    __init__.py             包初始化
    inputs.py               统一输入抽象（CSI/USB/视频/图片 + auto_select_source fallback）
    detector.py             YOLO 检测器（P0 四类定义、COCO 冷启动、异常降级、engine 预留）
    event_builder.py        事件聚合（连续帧检测、去抖、优先级、P1 硬隔离）
    pipeline.py             主链路编排（一帧一播仲裁、音频冷却、启动预加载）
    semantics.py            中文语义生成（<50 字提示、方向/距离格式化、28 条模板）
    audio_manager.py        音频管理（MiniMax->pyttsx3->print 降级链、全局锁、预加载）
    audio.py                音频底层（MiniMax TTS API、缓存、播放器适配）
    benchmark.py            性能基准（FPS/延迟采集、JSON/CSV 输出）
    video_demo.py           视频/图片序列演示入口
    usb_demo.py             USB 摄像头实时演示入口
    image_demo.py           单张图片演示入口
    demo.py                 Mock 数据演示（无摄像头，验证链路）
    publisher.py            结构化事件上报（HTTP POST，不上传原始图像）
    models.py               数据模型（DetectionEvent 等 dataclass）
    camera.py               [死代码] 字符串拼接工具，未被引用，待删除
    camera_check.py         摄像头检查工具（诊断 CSI/USB 状态）

datasets/                   数据集
    linkable_p0_raw/        自采原始数据目录
        blind_road_occupied/images/   盲道占用（目标 210 张，当前 0 张）
        stairs/images/                台阶（目标 120 张，当前 0 张）
        ramp/images/                  坡道（目标 120 张，当前 0 张）
        road_obstacle/images/         路面障碍（目标 150 张，当前 0 张）
        negative/images/              负样本（无障碍场景）
        rejected_privacy/             隐私审查不通过的样本
        manifest.csv                  采集记录表
    public_p0_sources/      公共数据源
        sources.json                6 个候选数据集元数据
        raw/                        未解压原始数据
        converted_yolo/             转换后 YOLO 格式
        _manifests/                 转换报告
    public_p0_yolo/         YOLO 训练数据集
        images/train/               训练集图片
        images/val/                 验证集图片
        images/test/                测试集图片
        labels/train/               训练集标注
        labels/val/                 验证集标注
        labels/test/                测试集标注
        data.yaml                   4 类定义文件

docs/                       项目文档
