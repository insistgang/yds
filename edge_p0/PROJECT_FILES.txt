LinkAble Edge P0 文件说明
==========================

最后更新：2026-05-25

本文件只保留工程入口索引。正式文案入口请优先看根目录 `DOCS_INDEX.md`。

核心源码
--------

`src/linkable_edge/`

- `inputs.py`：统一输入抽象，维护 CSI/USB/视频/图片序列 fallback。
- `detector.py`：YOLO 检测器，维护 COCO 冷启动、mangdaojiance 映射和 P0 自定义模型入口。
- `event_builder.py`：P0 事件聚合、连续帧去抖、P1 标签隔离。
- `pipeline.py`：检测结果到语音/上报的一帧处理主链路。
- `semantics.py`：P0/P1 模板化中文提示。
- `audio.py` / `audio_manager.py`：MiniMax、pyttsx3、print 降级链和音频缓存。
- `image_demo.py` / `video_demo.py` / `usb_demo.py` / `demo.py`：演示入口。
- `publisher.py`：结构化事件输出。
- `benchmark.py`：FPS、延迟统计。

当前脚本入口
------------

`scripts/`

- `capture_p0_frames.py`：现场数据采集。
- `check_p0_dataset.py`：原始自采数据检查。
- `prepare_public_p0_datasets.py`：公共数据转换。
- `convert_mangdaojiance_to_p0.py`：16 类盲道数据映射到 P0 四类。
- `merge_stairs_ramps_dataset.py`：stairs/ramp 数据集合并。
- `resplit_dataset.py`：P0 YOLO 数据集重新划分。
- `train_p0_model.py`：当前训练/评估/导出主入口。
- `visualize_p0_dataset.py`：数据可视化和统计。
- `select_poster_samples.py`：海报样例挑选。
- `generate_all_voice_alerts.py`：语音模板批量生成。
- `tts_minimax_test.py`：MiniMax TTS 合成测试。

归档脚本
--------

`scripts/archive/`

这里放一次性脚本或被主入口替代的脚本，不作为当前工作流入口。

数据与产物
----------

- `datasets/p0_yolo/`：当前融合后的 YOLO 数据集，体积较大，不进 Git。
- `pic/`：AI/临时图片素材，体积较大，不进 Git。
- `runs/`：训练、评估、可视化输出，不进 Git。
- `tts_samples/` / `voice_alerts/`：音频缓存和导出音频，按演示需要保留。
