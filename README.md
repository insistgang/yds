# yds — LinkAble（共感）

> **项目名称**: 基于边缘智能的无障碍辅助通行感知系统  
> **英文**: Edge Intelligence-Based Assistive Perception System for Accessible Navigation  
> **竞赛**: 第二十一届中国研究生电子设计竞赛 — 技术类竞赛 — 开放赛道 — 人工智能类 — 方向二：具身智能系统  
> **专项**: AI智能体专项赛  
> **截止日期**: **2026-06-20** 🔴

## 一句话定义

部署于人行无障碍空间关键节点的边缘智能系统，通过 YOLO 视觉识别盲道占用、台阶、坡道、路面障碍，提供实时中文语音播报，并沉淀结构化事件数据支撑无障碍治理。

## 核心定位

```
无人驾驶让车看懂路，LinkAble 让弱势行人听懂路、让管理者看见盲区。
```

**这不是**: 穿戴式导盲设备 / 手机 App / 云平台 / 志愿者系统  
**这是**: 固定节点 + 手持巡检的边缘感知原型系统

## 系统架构

```
摄像头/图片/视频输入
  → YOLO 检测（yolo11n.pt）
  → 事件聚合与去抖（EventBuilder）
  → 模板化中文提示（semantics.py）
  → 本地音频播报（audio.py / MiniMax TTS）
  → 结构化事件记录（publisher.py）
```

**P1 增强链路（离线）**:
```
结构化事件流 → 大模型分析（Kimi/Qwen）→ 周/月度无障碍报告
```

## P0 检测类别

|------|--------|--------|------|
| 台阶 | `stairs` | 100 | 安全风险最高 |

| 盲道占用 | `blind_road_occupied` | 70 | 盲道被占用 |
| 坡道 | `ramp` | 60 | 无障碍坡道 |
