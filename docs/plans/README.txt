docs/plans/ 目录说明
===================

本目录包含 LinkAble 项目的三个核心执行文档，均已对齐 PRD v2.2 / AGENTS.md。


1. linkable-competition-master-checklist.md
------------------------------------------
比赛版总清单。项目管理总纲。

主要内容：
- P0/P1/P2 范围冻结（什么必做、什么增强、什么不做）
- 隐私保护硬约束（不存原始视频、不上传图像流、匿名化）
- 量化验收标准（FPS>=30、延迟<100ms、响应<1.5s、识别率>=80%）
- 校园最后 20 米 A-D 点位演示路线
- 提交材料清单（论文/视频/展架/PPT/照片）
- Phase 执行计划（Phase 1-6 里程碑）
- 本周底线任务（数据采集目标：从 0 到 100+）

适用场景：团队每日站会、里程碑验收、提交前检查。


2. jetson-troubleshooting.md
---------------------------
Jetson 常见报错排查清单。现场故障手册。

主要内容：
- 刷卡后无法启动
- 系统卡顿/磁盘满
- PyTorch / CUDA 不可用
- Ultralytics 运行失败
- 项目 mock 链路中断
- CSI 相机不出帧（AGENTS.md 8.1 处理顺序）

适用场景：Jetson 现场出问题、赛前设备检查、远程 SSH 排查。


3. jetson-camera-diagnosis.md
------------------------------
相机出帧问题诊断记录。技术档案。

主要内容：
- 2026-04-27 IMX219 CSI 相机故障完整诊断
- 环境配置、错误现象、关键日志
- 已排除项（软件/驱动正常，指向硬件层）
- 处理结果：已按 AGENTS.md 8.1 切换至 USB 摄像头（验证通过）
- 当前可用输入源（CSI/USB/视频/图片四种 fallback）

适用场景：硬件问题复盘、向评审解释 CSI 故障处理、后续相机恢复参考。
