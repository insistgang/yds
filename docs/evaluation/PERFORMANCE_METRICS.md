## 🎯 模型性能（训练结果）

### 训练配置

- **模型**：YOLO11n
- **数据集**：17,337 张图片（训练集 14,909 + 验证集 1,560 + 测试集 868）
- **类别**：4 类（blind_road_occupied, stairs, ramp, road_obstacle）
- **训练轮次**：150 epochs
- **图片尺寸**：640x640

### 最终指标（Epoch 150）

| 指标 | 数值 |
|------|------|
| **Precision** | 0.8774 |
| **Recall** | 0.7684 |
| **mAP50** | 0.8264 |
| **mAP50-95** | 0.6336 |

### 训练曲线

- **Box Loss**：从 1.30 下降到 0.88
- **Class Loss**：从 2.04 下降到 0.58
- **DFL Loss**：从 1.38 下降到 1.11
